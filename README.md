# Int-Array-Designer
A menu-driven Python program that lets users manage an integer array by adding numbers, deleting maximum/minimum/specific values, reordering ascending or descending, and checking the current array state.
# Tools Used
- Python 3.x
- CLI (menu-driven) interface
